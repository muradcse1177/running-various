Here is the steps to translate the theme, if you still need any help just let me know
First you need to install http://poedit.net/ then follow these steps.
1- first you need to get your language locale, you can find the list here https://make.wordpress.org/polyglots/teams/ for example German locale is de_DE (we will have de_DE as an example for the next steps, and you need to make sure that you are using Wordpress version in the same language you need to translate to)
2- go to theme folder > Languages folder and open the file default.pot by the application you just installed (Poedit).
3- then select File > save as and name the file de_DE and store it in the same folder Sanabel > Languages
4- this will create 2 files de_DE.mo and de_DE.po
5- both of them are important but you use de_DE.po to translate strings.
6- open de_DE.po by the same application and you will find a list of all words used in the theme.
7- click the word you want to translate and write its translation in Translation field then save the file
