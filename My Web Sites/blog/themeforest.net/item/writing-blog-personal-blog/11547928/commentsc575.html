A-Support Author Team

over 1 year ago
Hi bro, thanks for purchasing Writing theme Im Heba a new team member and Im here to help you. To center the logo you need to add Css code, I recommend this plugin https://wordpress.org/plugins/simple-custom-css/ and you can add the code through Appearance > Custom Css Heres the css code: .logo_wrapper { width: auto; float: none; text-align: center; } .site_logo, .site_logo a { display: inline-block; float: none!important; } and if you have a sliding sidebar enabled, you will need to center its icon too, so you need to add these lines .header_info_wrapper { width: auto; float: none; text-align: center; } .user_info_button { display: inline-block; float: none; margin: 0; }

and the ideal logo dimensions is max-width: 450450 but it varies depending on the logo original dimensions. Enjoy using Writing theme and feel free to ask at anytime
